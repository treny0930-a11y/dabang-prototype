# 다방 To-be v3 — UX 리디자인 프로토타입

부동산 앱 다방의 핵심 문제(허위매물·신뢰 부재)를 해결하는 UX 개선안 인터랙티브 프로토타입.

**Play Store 1★ 리뷰 12,455개 분석** → 신뢰 지표를 매물 리스트 단계부터 노출하는 방식으로 개선.

## 🔗 라이브 데모

👉 **[dabang-prototype.vercel.app](https://dabang-prototype.vercel.app)**

## 화면 구성

| 화면 | 내용 |
|------|------|
| S1 홈 | 안심배너 + 신뢰지표 매물 카드 |
| S2 매물리스트 | 안심점수·응답률 필터 카드 |
| S3 매물상세 | 안심점수 링 (87점) + 하트/공유 인터랙션 |
| S4 가격정보 | 시세비교 분석 배너 |
| S5 상세정보 | 위반건축물 하이라이트 |
| S6 옵션/안심 | 안심 체크리스트 카드 |
| S7 거래정보 | 시세비교 ±% 컬러코딩 |
| S8 3D투어 | 현상 유지 |
| S9 중개사 | 응답률·응답시간·리뷰 + 채팅 프리뷰 |
| S10 검색 | 최근검색·빠른검색·인기지역 |

## 핵심 UX 개선

- **기존:** 신뢰 정보가 매물 상세 5번째 탭 이후에야 노출
- **개선:** 안심점수·응답률·응답시간을 매물 리스트 카드 단계부터 노출

## 기술 스택

- HTML / CSS / Vanilla JS (단일 파일)
- Pretendard Variable 폰트
- Vercel 정적 배포
